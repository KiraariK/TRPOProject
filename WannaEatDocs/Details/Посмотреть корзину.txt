Имя прецедента	Посмотреть корзину
Идентификатор прецедента	ID хх
Краткое описание	Данный прецедент позволяет пользователю оформить заказ.
Актеры, вовлеченные в прецедент
Главные актеры: Клиент
Второстепенные актеры: -
Состояние системы до начала прецедента	Предусловия:
1.	Выбран город
2.	Выбрано заведение
Фактические этапы прецедента
1.	Прецедент начинается, когда Клиент выбирает опцию «Посмотреть заказ (корзину)».
2.	Для каждого блюда в корзине:
2.1.	Система выводит наименование, количество, изображение, цену.
3.	Система выводит общую стоимость блюд в корзине.
4.	Клиент выбирает «Оформить заказ».
5.	Система формирует заказ.
6.	Клиент выбирает «Заказать столик».
Точка расширения «Table order».
Состояние системы после окончания прецедента	
Постусловия: -
Альтернативные потоки
1.	«Очистка корзины».
2.	«Удалить пункт из заказа».
3.	Клиент выбирает «Заказать самовывоз» и вводит время.
Точка расширения «Pickup order».
4.	Клиент выбирает «Заказать доставку» и вводит время.
Точка расширения «Delivery order».
